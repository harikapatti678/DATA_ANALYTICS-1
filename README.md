# DATA_ANALYTICS-1
A Data Science and Machine Learning project that analyzes the Netflix Titles dataset using Python, performs data cleaning, exploratory data analysis (EDA), visualization, and builds a Random Forest classifier to predict content type (Movie/TV Show).

# 🎬 Netflix Data Analysis & Machine Learning

> Transforming raw Netflix data into meaningful insights through Data Science and Machine Learning.

---

## 📖 Project Overview

This project explores the Netflix Titles dataset to uncover valuable insights about movies and TV shows available on the platform. Using Python and Machine Learning, the dataset is cleaned, analyzed, visualized, and used to build a predictive model. The project follows a complete Data Science workflow from data preprocessing to model evaluation.

---

## 🎯 Objectives

- Clean and preprocess real-world Netflix data.
- Handle missing values and improve data quality.
- Perform Exploratory Data Analysis (EDA).
- Visualize trends and patterns in Netflix content.
- Build a Machine Learning classification model.
- Evaluate model performance using standard metrics.

---

## 📂 Dataset Information

- **Dataset:** Netflix Titles
- **Source:** Kaggle
- **Format:** CSV
- **Records:** 8,800+ Rows
- **Attributes:** 12 Columns

The dataset contains details such as title, director, cast, country, release year, rating, duration, genre, and type (Movie/TV Show).

## 🔄 Project Workflow

### 1️⃣ Data Collection
- Imported the Netflix dataset from CSV format.

### 2️⃣ Data Cleaning
- Identified missing values.
- Replaced null values where necessary.
- Removed inconsistencies.

### 3️⃣ Exploratory Data Analysis (EDA)
- Distribution of Movies vs TV Shows.
- Most common content ratings.
- Top content-producing countries.
- Release year trends.
- Content growth over time.

### 4️⃣ Data Visualization
Created informative visualizations including:
- Bar Charts
- Line Charts
- Frequency Analysis

### 5️⃣ Feature Engineering
- Selected relevant features.
- Encoded categorical variables using Label Encoding.

### 6️⃣ Machine Learning
Developed a **Random Forest Classifier** to classify Netflix content based on selected features.

### 7️⃣ Model Evaluation
Evaluated the model using:
- Accuracy Score
- Confusion Matrix
- Classification Report

---

## 📊 Key Insights

✔ Netflix contains significantly more Movies than TV Shows.

✔ Content production has increased rapidly after 2015.

✔ Certain countries contribute the majority of Netflix content.

✔ TV ratings vary depending on content type and target audience.

✔ Machine Learning can effectively classify content using structured features.
